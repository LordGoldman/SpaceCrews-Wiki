# SpaceCrews

**Social deduction for Minecraft.** Crewmates complete tasks; impostors kill, sabotage,
and vent. Meetings, voting, ejections, special roles, Hide n Seek — the full loop, on
your own maps, for Spigot/Paper **1.16 → latest**.

## Highlights

<div class="grid cards" markdown>

- **60+ task minigames** — wiring, swipe card, medbay scan, uploads, asteroids…
  common/short/long assignment, fake lists for impostors, visual tasks, task-bar win.
- **Full sabotage suite** — lights (shared 5-switch panel), reactor & O2 criticals,
  comms blackout, region doors, decontamination corridors.
- **Info systems** — security cameras (teleport view with a killable body double),
  admin table, vitals, MIRA-style door log.
- **9 special roles** — Scientist, Engineer, Noisemaker, Guardian Angel, Viper,
  Tracker, Detective, Phantom, Shapeshifter — plus the Mushroom Mixup.
- **Crew-color skins** — packet-based colored crewmate skins, per-viewer vision
  culling: nametags never leak through walls, deaths stay secret until meetings.
- **Map transit** — vents, ziplines, an Airship-style gap platform, spawn picking.
- **Hide n Seek** — the official alternate mode, on the same maps.
- **Parties, browser, per-party settings** — 30+ host-adjustable options in a GUI.
- **Stats, XP & levels** — SQLite or shared MySQL, PlaceholderAPI support.
- **Network mode** — lobby + game servers over Redis: cross-server browser,
  reservations, rejoin grace, autoscaling.
- **Developer API** — clean api jar, events (incl. cancellable pre-events), custom
  task minigames.

</div>
