# Hide n Seek

The official alternate mode, on the same maps. Toggle **Game Mode** in the party
settings (host); the mode reshapes the rules:

## Rules

- Exactly **one Seeker** (the impostor) — publicly revealed at start, then frozen
  for a 10-second head start while hiders scatter.
- **Hiding phase** (default 200s): hiders survive. Each hider gets a **danger
  meter** action bar — a proximity readout to the Seeker (red when close).
- Completing tasks **shaves time off the clock** (task bonus setting) — hiding
  passively is viable, doing tasks ends it faster.
- **Final hide** (default 50s): the Seeker gets a speed boost, tasks lock, and
  flame pings mark every living hider's position for the Seeker at intervals.
- Timer runs out → **hiders win**. Seeker catches everyone → **Seeker wins**.

## What changes vs classic

| System | Hide n Seek |
|---|---|
| Meetings, reports, emergency button | disabled |
| Sabotage (incl. doors) | disabled |
| Special roles | disabled |
| Kill cooldown | fixed 1 second (the chase is the game) |
| Vents | flipped: **hiders** get limited "Hide" uses (max stay applies); the Seeker can't vent |
| Task bar | replaced by the survival clock |

## Host knobs

Hiding time, final-hide time, task time bonus, Seeker speed multiplier, hider vent
uses, vent max stay, and the final-phase ping interval — all in the settings GUI.
